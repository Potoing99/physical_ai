# 작업 구분 안내 (physical_ai)

이 저장소는 수업 작업을 번호별로 나누어 관리합니다.

- **1번: 스크래핑 앱**
  - 폴더: `1번_스크래핑앱/`
  - 설명: 인터넷 정보 스크래핑 결과를 출력하는 앱

- **2번: CSV 추이 그래프**
  - 폴더: `2번_CSV그래프/`
  - 설명: CSV 파일을 읽어 데이터 추이를 그래프로 출력

- **3번: 자동 작업 + 블로그 문서**
  - 폴더: `3번_자동작업_블로그/`
  - 설명: 하위 폴더 생성, `index.md`(3편 목차), `post1.md`(1편 초안) 작성

## 사용 원칙
- 작업은 해당 번호 폴더 안에서 진행
- 작업 완료 시 Git 커밋/푸시로 이력 관리
