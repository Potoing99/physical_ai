# 작업 구분 안내 (physical_ai)

이 저장소는 수업 작업을 번호별로 나누어 관리합니다. (Python 기준)

- **1번: 스크래핑 앱 (Python)**
  - 폴더: `1번_스크래핑앱/`
  - 실행: `python app.py`

- **2번: CSV 추이 그래프 (Python)**
  - 폴더: `2번_CSV그래프/`
  - 실행: `python plot_csv_trend.py`

- **3번: 자동 작업 + 블로그 문서 (Python)**
  - 폴더: `3번_자동작업_블로그/`
  - 실행: `python auto_blog_task.py`

## 사용 원칙
- 작업은 해당 번호 폴더 안에서 진행
- 작업 완료 시 Git 커밋/푸시로 이력 관리
